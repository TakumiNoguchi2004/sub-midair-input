# Mid-Air Flick input システム

## 絵文字入力システム


- OpenMoji（hfg-gmuend/openmoji）を入力データとし、学習済み CLIPなどのような画像とテキストを共通の特徴空間に写像できるモデルを用いて検索を行うシステムを作る。

- OpenMojiから取得した、emoji(の画像)データを、FAISSを用いてファイルベースのベクトルDBとして保存しておき、入力に適応する絵文字画像を検索して返すシステムになる。

- 入力は主に2種類ある。
  - テキスト検索
    - テキストをText-Encoder によってベクトルにする。
  - 手書き入力 -> 画像化して画像検索
    - この入力方法を選択した場合、まずwindow が立ち上がる。ここにマウスクリックとドラッグで絵文字を書き、「入力」としたらその入力した絵文字を画像として扱い、Image-Endoer によってベクトル化する。

- 入力の際、「上位N件を取り出す」と言った指定ができるようにする。いったん実行時引数として、 `--top-k 10`のようにできれば良いと思う。
